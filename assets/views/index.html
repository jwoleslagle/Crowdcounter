Welcome!

Crowdcounter is a simple project which attempts to count the number of people in a snapshot of a crowd using Amazon's Rekognition facial recognition service.

It has some limitations...

  • Images must be stored to be analyzed. Storage is cheap, but not free, so sign-in is required.
  • Ideally, the crowd in the picture should be facing the camera with good, flat light (think of your 3rd grade class picture).
  • Even then, it might not work very well! With Rekognition I'm seeing about 30-70% of faces recognized - your mileage may vary. I hope to make this a platform to try out different visual recognition solutions.
  • At present, Rekognition can only spot a maximum of 100 people per image.
  • I have a wishlist a mile long for this project, but the primary goal was to learn the basics of Node.js, Express, AWS S3 and Rekognition, and MongoDb. As such, if you spot any issues, feedback (via the GitHub page) is always appreciated.
